Meta Tags

  • São lidas somente pelo navegador
  • Ficam dentro do <head>

  • Atributos do Meta Tags:

    • <meta charset="UTF-8"> - Define a codificação de caracteres do documento HTML, nesse caso, o UTF-8
    • <meta name="description"> - Breve descrição da página, usada por motores de busca para exibir snippet. - EX: <meta name="description" content="Tutoriais gratuitos de HTML e CSS">
    • <meta name="viewport"> - Configura o viewport para responsividade em dispositivos móveis. - EX: <meta name="viewport" content="width=device-width, initial-scale=1.0">
    • <meta name="author"> - Informa o autor do documento. - EX: <meta name="author" content="João Silva">
    • <meta name="keyword"> - Lista de palavras-chave relevantes para o conteúdo da página. - EX: <meta name="keywords" content="HTML, CSS, JavaScript">

Tags de Cabeçalho e Paragrafo

  • Tags de Cabeçalho são tags de título. Elas vão de 1 até 6, onde quanto menor o número, menor o tamanho e importancia.
  • As Tags de Cabeçalho organizam títulos e subtítulos em diferentes níveis de hierarquia.

  • Tipos de Tags de título

    1. <h1> - Título principal da página (ex: nome do site)
    2. <h2> - Subtítulo ou seção principal
    3. <h3> - Subtítulo de uma seção
    4. <h4> - Subtítulo de uma subseção
    5. <h5> - Subtítulo de uma subseção menor
    6. <h6> - Subtítulo de uma subseção ainda menor

  • A Tag de Paragrafo é o <p> - Define parágrafos.

Tags de formatação

  • Negito - <b>(somente visual) ou <strong>(informa ao navegador que a palavra tem um grau a mais de importancia). EX: Eu gosto de bolo!
  • Linha de divisão - <hr>.
  • Quebra de Linha - <br>.
  • Itálico - <i> ou <em>. EX: Eu gosto de bolo!
  • Sublinhado - <u>. EX: Eu gosto de bolo!
  • Diminuir texto - <small>. EX: Eu gosto de bolinho!
  • Riscado - <del>. EX: Eu gosto de bolo pirulito!
  • A cima - <sup>. EX: O bolo custa R$ 3,003
  • A baixo - <sub>. EX: O bolo está na posição X12 da prateleira
  • Marcado - <mask>. EX: Eu gosto muito de bolo!

Elementos de Citação

  • Abreviação - <abbr>. EX: Eu estouaprendento HTML
  • Enderenço - <address>. Pode ser físico ou digital. EX: Você pode entrar em contato pelo número de telefone ou email:
    90000-0000, joao@email.com
  • Título de uma obra - <cite>. EX: Ele já leu Sherlock Holmes?
  • Citação curta - <q>. EX: Só sei que nada sei. (Sócrates)
  • Citação longa - <blockquote>. Tem um recuo. EX:
    Só sei que nada sei. (Sócrates)
  • Inverter - <bdo dir="rtl">. Tem um recuo. EX: A imagem no espelho da palavra "manteiga" é manteiga.

Hiper links

<a href="O link do site">

  • link(URL absoluta) - <a href="O link do site">. EX: Acessar vídeos sobre links
  • link(URL absoluta), mas sendo aberto em uma página diferente - <a href="O link do site" target="_blank>". EX: Acessar vídeos sobre links
  • link(URL relativa) - <a href="O link do site">. EX: Acessar arquivo SiteSobreAtividade.html

Tabelas

  • <table> - Cria a tabela.
  • <tr> - Linha da tabela.
  • <th> - Coluna de tópicos da tabela. EX: Nome, Preço, tamanho, peso.
  • <td> - Coluna de elementos da tabela. EX: João, R$ 7,00, 5m², 70kg.

  • Atributos das tabelas:

    • <border> - Espessura da borda da tabela. EX: <table border="1">
    • <cellpadding> - Espaço interno entre conteúdo e borda. EX: <table cellpadding="5">
    • <cellspacing> - Espaço entre as células. EX: <table cellspacing="2">
    • <cellspacing> - Espaço entre as células. EX: <table cellspacing="2">
    • <width> - Largura da tabela (px ou %). EX: <table width="80%">
    • <height> - Altura da tabela (px ou %). EX: <table height="200">
    • <align> - Alinhamento da tabela na página (recomendável usar CSS). EX: <table align="center">
    • <bgcolor> - Cor de fundo da tabela (antigo e é recomendável usar CSS). EX: <table bgcolor="#eee">

    Exemplo de tabela:

    Produto Preço Tipo
    Esponja R$ 6,00 Produto de limpeza
    Bolo R$ 25,00 Allimento
    Conjunto de pratos R$ 58,00 Utensílio de cozinha

Listas

  • <ul> - Lista não ordenada.
  • <ol> - Lista ordenada.
  • <li> - Item da lista.
  • <dl> - Lista de definição.
  • <dt> - Termo a ser definido.
  • <dd> - Definição do termo.

  • Exemplo de lista não ordenada:

    • Item 1
    • Item 2
    • Item 3

  • Exemplo de lista ordenada:

    1. Item 1
    2. Item 2
    3. Item 3

Formulário
  • <form> - Elemento que define o formulário. Possui atributos importantes como:
    action: URL para onde os dados do formulário serão enviados.
    method: Método HTTP para envio dos dados, geralmente "GET" ou "POST".
  • <label> - Rótulo que descreve o propósito de um campo, facilitando a usabilidade e acessibilidade. Atributos do label:

    • for - Associa o rótulo a um elemento de formulário específico, vinculando o texto do <label> ao elemento cujo id corresponde ao valor desse atributo.
    • id - Identificador único para o próprio elemento <label>, permitindo que ele seja referenciado via CSS ou JavaScript.
    • class - Permite atribuir uma ou mais classes CSS ao <label>, facilitando a estilização em massa.
    • style - Adiciona estilos CSS diretamente ao elemento <label>.
    • title - Fornece um texto alternativo que aparece como dica (tooltip) ao passar o mouse sobre o rótulo.
    • lang - Define o idioma do conteúdo do <label>
    • dir - Especifica a direção do texto (por exemplo, da esquerda para a direita ou da direita para a esquerda).

  • <input> - Campo de entrada de dados, que pode ser configurado com diferentes tipos, como texto, email, senha, número, entre outros. Atributos do input:

    • type - Define o tipo de controle do campo (texto, senha, e-mail, data, etc.). Exemplos: text, password, email, date, checkbox, radio, file.
    • name - Identifica o campo ao enviar o formulário, usado para processamento no servidor.
    • value - Define o valor inicial/padrão do campo. Para botões, define o texto exibido.
    • placeholder - Exibe uma dica contextual dentro do campo (ex: "Digite seu e-mail").
    • required - Torna o campo obrigatório para envio do formulário.

  • <textarea>- Área de texto para inserção de múltiplas linhas, ideal para mensagens ou descrições longas
  • <button> - Botão que pode ser usado para enviar o formulário ou executar outras ações
  • <select> e <option> - Elementos para criar listas suspensas (dropdowns) com opções selecionáveis
  • <fieldset> e <legend > - Usados para agrupar campos relacionados e dar um título a esse grupo, melhorando a organização visual e semântica do formulário
  • <output> - Elemento para exibir resultados ou saídas relacionadas ao formulário

Áudio

  • O <audio> embute conteúdo sonoro em páginas HTML.
  • O <audio> usa <source> para referenciar arquivos de áudio com seus tipos.
  • Possui atributos para controle da reprodução.

Atributos do áudio:

  • controls: Exibe os controles padrão do player (play, pause, volume).
  • autoplay: Faz o áudio iniciar automaticamente ao carregar a página.
  • loop: Faz o áudio repetir indefinidamente.
  • preload: Indica se o áudio deve ser carregado antes da reprodução (opções: none, metadata, auto).
