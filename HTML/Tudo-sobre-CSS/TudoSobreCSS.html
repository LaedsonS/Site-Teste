Seletores e Classes

Seletores
Seletores são padrões usados para identificar quais elementos HTML serão estilizados pelo CSS. Eles permitem direcionar elementos específicos na página para aplicar regras de estilo. Existem vários tipos de seletores, incluindo:
Seletores de tipo
Selecionam todos os elementos de um determinado tipo, como todos os <p> ou <h1>.
Seletores de ID
Selecionam um elemento único com um ID específico.
No HTML: <p id="NoCentro">
No CSS: #NoCentro{ text-aling: center;}
Seletores de classe ou Classes
Classes são atributos HTML que podem ser atribuídos a um ou vários elementos para categorizá-los ou agrupá-los para aplicação de estilos comuns. Elas são definidas no HTML usando o atributo class e referenciadas no CSS com um ponto (.) seguido do nome da classe.
No HTML: <h2 class="Vermelho">
No CSS: .vermelho{ color: vermolho;}

Propriedade Color

A propriedade color no CSS é usada para definir a cor do texto de um elemento HTML, incluindo seu conteúdo textual e decorações como sublinhados ou sombras de texto. Ela afeta diretamente a cor dos caracteres dentro do elemento ao qual é aplicada.

A sintaxe básica da propriedade é:

color: valor;

Onde o valor pode ser:

  • Nome da cor (ex: red, blue, rebeccapurple)
  • Código hexadecimal (ex: #ff0000, #090)
  • Valores RGB ou RGBA (ex: rgb(255, 0, 0), rgba(255, 0, 0, 0.5))
  • Valores HSL ou HSLA (ex: hsl(0, 100%, 50%), hsla(0, 100%, 50%, 0.5))
  • Valores globais como inherit, initial e unset

Propriedade Background

A propriedade background no CSS é uma forma abreviada (shorthand) que permite definir várias propriedades relacionadas ao plano de fundo de um elemento em uma única declaração. Ela pode incluir valores para cor, imagem, posição, repetição, tamanho, origem, recorte e fixação do fundo, entre outros aspectos.

Propriedades que a shorthand background pode definir

  • background-color: define a cor de fundo do elemento.
  • background-image: define uma ou mais imagens de fundo.
  • background-repeat: controla se e como a imagem de fundo se repete.
  • background-attachment: determina se a imagem de fundo é fixa ou rola com o conteúdo.
  • background-position: define a posição inicial da imagem de fundo.
  • background-size: especifica o tamanho da imagem de fundo.
  • background-origin: define a área do box onde a imagem começa a ser posicionada.
  • background-clip: especifica a área do box onde o fundo é pintado.

Propriedade Border

A propriedade border no CSS é uma forma abreviada (shorthand) que permite definir a borda de um elemento HTML, incluindo sua largura, estilo e cor, em uma única declaração.

Componentes principais da propriedade border

  • border-width: define a espessura da borda. Pode ser especificada em unidades como pixels (px), em, rem, vh, vw, ou por palavras-chave como thin (1px), medium (3px) e thick (5px).
  • border-style: determina o estilo da borda. Os valores mais comuns são:
    • none (sem borda)
    • solid (linha contínua)
    • dotted (pontilhada)
    • dashed (tracejada)
    • double (linha dupla)
    • Outros estilos como groove, ridge, inset e outset também são suportados

  • border-color: define a cor da borda, que pode ser especificada por nome (ex: red), código hexadecimal (ex: #000000), RGB, HSL, etc.
  • border-radius: arredonda os cantos das bordas de um elemento.

  • Propriedades relacionadas para bordas específicas: Além da propriedade geral border, é possível definir bordas individualmente para cada lado do elemento:
    • border-top
    • border-right
    • border-bottom
    • border-left

A borda faz parte do modelo de caixa (box model) do CSS, posicionada entre o padding e a margin do elemento.
A borda padrão é none, ou seja, invisível, até que seja definida uma largura e um estilo.
É possível usar a propriedade border para estilizar bordas com imagens, embora isso envolva propriedades específicas adicionais.

A propriedade border aceita até três valores na seguinte ordem: border: <border-width> <border-style> <border-color>;

Propriedade Marginr

A propriedade margin no CSS define a área de margem externa de um elemento, ou seja, o espaço em volta dele, fora de qualquer borda. Ela pode ser usada para criar espaçamento entre elementos na página.

Como funciona a propriedade margin

  • margin é uma propriedade abreviada (shorthand) que pode definir as margens dos quatro lados do elemento:

    • margin-top
    • margin-bottom
    • margin-right
    • margin-left

  • A sintaxe pode receber de 1 a 4 valores, que são aplicados conforme a ordem:

    • 1 valor: aplica a mesma margem para os quatro lados.
    • 2 valores: o primeiro para as margens superior e inferior, o segundo para as margens direita e esquerda.
    • 3 valores: o primeiro para a margem superior, o segundo para as margens direita e esquerda, o terceiro para a margem inferior.
    • 4 valores: aplicados na ordem top, right, bottom, left (sentido horário).
    • Exemplo: margin: 10px 50px 20px 0; /* topo 10px, direita 50px, base 20px, esquerda 0 */

  • Margens criam espaço transparente e não podem ser preenchidas com cor nem receber bordas.
  • Elementos inline não aceitam margin-top e margin-bottom, apenas elementos block ou inline-block.
  • Margens adjacentes entre elementos podem se "fundir" (colapso de margem), prevalecendo a maior margem.
  • Valores negativos podem ser usados para sobrepor elementos, mas devem ser usados com cuidado.

Propriedade Padding

A propriedade padding no CSS define o espaçamento interno entre o conteúdo de um elemento e suas bordas, criando um espaço "de dentro" que afasta o conteúdo das bordas do elemento.
É uma propriedade abreviada que pode definir os quatro lados do preenchimento: superior, direito, inferior e esquerdo.

Como funciona a propriedade padding

  • Pode ser usada com 1 a 4 valores, aplicados na ordem: top (superior), right (direito), bottom (inferior), left (esquerdo). Exemplo:
    • padding: 20px; - aplica 20px em todos os lados.
    • padding: 10px 15px; - 10px no topo e base, 15px nas laterais.
    • padding: 5px 10px 15px; - 5px no topo, 10px nas laterais, 15px na base.
    • padding: 5px 10px 15px 20px; - valores específicos para cada lado, no sentido horário começando pelo topo.
  • Também é possível definir o padding individualmente com: padding-top, padding-right, padding-bottom, padding-left.
  • Valores negativos não são permitidos para padding.
  • O padding aumenta o tamanho total do elemento, pois é somado à largura e altura definidas, a menos que se use box-sizing: border-box para incluir o padding dentro do tamanho total do elemento.

Criando um botão simples com padding

.botao {
text-decoration: none;
padding:20px;
margin-bottom: 70px;
border-style: double;
border-radius: 50px;
border-color: black;
color: rgb(27, 79, 70);
background-color: rgb(227, 173, 58);
}
.botao:hover {
color:blue;
background-color: violet;
}

Propriedade Width e Height

As propriedades width e height são usadas para definir, respectivamente, a largura e a altura da área de conteúdo de um elemento, principalmente em CSS e programação visual.

Propriedade Width

  • Define a largura da área de conteúdo de um elemento, excluindo margens, bordas e preenchimentos (padding).
  • Pode ser especificada em valores absolutos (pixels, em, rem, etc.), percentuais (em relação ao elemento pai) ou como auto para ajuste automático.
  • As propriedades min-width e max-width podem limitar a largura mínima e máxima, sobrescrevendo o valor de width.
  • Aplica-se a quase todos os elementos, exceto alguns inline e elementos de tabela específicos.

Propriedade Height

  • Define a altura da área de conteúdo de um elemento, também excluindo margens, bordas e preenchimentos.
  • Pode ser especificada em unidades absolutas, percentuais (em relação à altura do elemento pai, se definida) ou como auto.
  • As propriedades min-height e max-height limitam a altura mínima e máxima, sobrepondo o valor de height.
  • Aplica-se a quase todos os elementos, exceto alguns inline e elementos de tabela específicos.

Propriedade Text

A propriedade text em CSS não é uma propriedade única, mas o termo geralmente se refere ao conjunto de propriedades usadas para estilizar textos em páginas web. Essas propriedades controlam a aparência, formatação e disposição do texto.

  • text-aling: Controla como o texto (ou outros conteúdos inline) é alinhado na horizontal dentro do seu contêiner. Pode ser: left(clinha o texto à esquerda), right(clinha o texto à direita), center(centraliza o texto horizontalmente dentro do elemento), justify(Estica as linhas do texto para que as margens esquerda e direita fiquem alinhadas), inherit(O elemento herda o valor de alinhamento do seu elemento pai).
  • direction: Afeta o fluxo do texto e também pode influenciar o layout de elementos, como em contêineres flexíveis, onde a direção do eixo principal pode ser afetada pelo valor de direction. pode ser: ltr ou rtl
  • unicone-bide: Corrige a exibição de textos que misturam idiomas com direções diferentes. Permite sobrescrever o algoritmo padrão do Unicode para manipulação de texto bidirecional, definindo níveis de incorporação e isolamento.
  • color: Define a cor do texto.
  • font-size: Define o tamanho da fonte, podendo usar unidades como px, em, %, pt.
  • text-decoration: Aplica decorações ao texto, como underline (sublinhado), overline (linha acima), line-through (riscado) ou none (sem decoração).
  • text-transform: Altera a capitalização do texto, com valores como uppercase (maiúsculas), lowercase (minúsculas) e capitalize (primeira letra maiúscula).
  • text-indent: Define o espaçamento da primeira linha de uma paragráfo.
  • word-spacing: Ajusta o espaçamento entre palavras.
  • letter-spacing: Ajusta o espaçamento entre letras. Pode ser negativo, fazendo com que as letras se sobrepõem
  • line-height: Ajusta o espaçamento entre linhas. Pode ser negativo, fazendo com que as linhas se sobrepõem

Propriedade Font

responsável por definir a formatação das fontes utilizadas em uma página web. Ela pode ser usada para configurar várias características do texto, como família, tamanho, estilo e variante da fonte.

  • font-style: Define o estilo da fonte, podendo ser normal (padrão), italic (itálico) ou oblique (obliquo).
  • font-size: Define o tamanho da fonte, aceitando unidades como px, em, %, entre outras.
  • font-weight: Controla o peso da fonte, como normal, negrito, etc.
  • font-family: : Especifica a família da fonte a ser usada. Pode incluir uma lista de fontes em ordem de prioridade, terminando geralmente com uma família genérica como serif, sans-serif ou monospace para garantir uma fonte alternativa caso as anteriores não estejam disponíveis no sistema do usuário.
