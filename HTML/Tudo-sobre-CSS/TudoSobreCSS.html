Seletores e Classes

Seletores
Seletores são padrões usados para identificar quais elementos HTML serão estilizados pelo CSS. Eles permitem direcionar elementos específicos na página para aplicar regras de estilo. Existem vários tipos de seletores, incluindo:
Seletores de tipo
Selecionam todos os elementos de um determinado tipo, como todos os <p> ou <h1>.
Seletores de ID
Selecionam um elemento único com um ID específico.
No HTML: <p id="NoCentro">
No CSS: #NoCentro{ text-aling: center;}
Seletores de classe ou Classes
Classes são atributos HTML que podem ser atribuídos a um ou vários elementos para categorizá-los ou agrupá-los para aplicação de estilos comuns. Elas são definidas no HTML usando o atributo class e referenciadas no CSS com um ponto (.) seguido do nome da classe.
No HTML: <h2 class="Vermelho">
No CSS: .vermelho{ color: vermolho;}

Propriedade Color

A propriedade color no CSS é usada para definir a cor do texto de um elemento HTML, incluindo seu conteúdo textual e decorações como sublinhados ou sombras de texto. Ela afeta diretamente a cor dos caracteres dentro do elemento ao qual é aplicada.

A sintaxe básica da propriedade é:

color: valor;

Onde o valor pode ser:

  • Nome da cor (ex: red, blue, rebeccapurple)
  • Código hexadecimal (ex: #ff0000, #090)
  • Valores RGB ou RGBA (ex: rgb(255, 0, 0), rgba(255, 0, 0, 0.5))
  • Valores HSL ou HSLA (ex: hsl(0, 100%, 50%), hsla(0, 100%, 50%, 0.5))
  • Valores globais como inherit, initial e unset

Propriedade Background

A propriedade background no CSS é uma forma abreviada (shorthand) que permite definir várias propriedades relacionadas ao plano de fundo de um elemento em uma única declaração. Ela pode incluir valores para cor, imagem, posição, repetição, tamanho, origem, recorte e fixação do fundo, entre outros aspectos.

Propriedades que a shorthand background pode definir

background-color:
Define a cor de fundo do elemento.
background-image:
Define uma ou mais imagens de fundo.
background-repeat:
Controla se e como a imagem de fundo se repete.
background-attachment:
Determina se a imagem de fundo é fixa ou rola com o conteúdo
background-position:
Define a posição inicial da imagem de fundo.
background-size:
Especifica o tamanho da imagem de fundo
background-origin:
Define a área do box onde a imagem começa a ser posicionada.
background-clip:
Especifica a área do box onde o fundo é pintado.

Propriedade Border

A propriedade border no CSS é uma forma abreviada (shorthand) que permite definir a borda de um elemento HTML, incluindo sua largura, estilo e cor, em uma única declaração.

Componentes principais da propriedade border

  • border-width: define a espessura da borda. Pode ser especificada em unidades como pixels (px), em, rem, vh, vw, ou por palavras-chave como thin (1px), medium (3px) e thick (5px).
  • border-style: determina o estilo da borda. Os valores mais comuns são:
    • none (sem borda)
    • solid (linha contínua)
    • dotted (pontilhada)
    • dashed (tracejada)
    • double (linha dupla)
    • Outros estilos como groove, ridge, inset e outset também são suportados

  • border-color: define a cor da borda, que pode ser especificada por nome (ex: red), código hexadecimal (ex: #000000), RGB, HSL, etc.
  • border-radius: arredonda os cantos das bordas de um elemento.

  • Propriedades relacionadas para bordas específicas: Além da propriedade geral border, é possível definir bordas individualmente para cada lado do elemento:
    • border-top
    • border-right
    • border-bottom
    • border-left

A borda faz parte do modelo de caixa (box model) do CSS, posicionada entre o padding e a margin do elemento.
A borda padrão é none, ou seja, invisível, até que seja definida uma largura e um estilo.
É possível usar a propriedade border para estilizar bordas com imagens, embora isso envolva propriedades específicas adicionais.

A propriedade border aceita até três valores na seguinte ordem: border: <border-width> <border-style> <border-color>;

Propriedade Marginr

A propriedade margin no CSS define a área de margem externa de um elemento, ou seja, o espaço em volta dele, fora de qualquer borda. Ela pode ser usada para criar espaçamento entre elementos na página.

Como funciona a propriedade margin

  • margin é uma propriedade abreviada (shorthand) que pode definir as margens dos quatro lados do elemento:

    • margin-top
    • margin-bottom
    • margin-right
    • margin-left

  • A sintaxe pode receber de 1 a 4 valores, que são aplicados conforme a ordem:

    • 1 valor: aplica a mesma margem para os quatro lados.
    • 2 valores: o primeiro para as margens superior e inferior, o segundo para as margens direita e esquerda.
    • 3 valores: o primeiro para a margem superior, o segundo para as margens direita e esquerda, o terceiro para a margem inferior.
    • 4 valores: aplicados na ordem top, right, bottom, left (sentido horário).
    • Exemplo: margin: 10px 50px 20px 0; /* topo 10px, direita 50px, base 20px, esquerda 0 */

  • Margens criam espaço transparente e não podem ser preenchidas com cor nem receber bordas.
  • Elementos inline não aceitam margin-top e margin-bottom, apenas elementos block ou inline-block.
  • Margens adjacentes entre elementos podem se "fundir" (colapso de margem), prevalecendo a maior margem.
  • Valores negativos podem ser usados para sobrepor elementos, mas devem ser usados com cuidado.

Propriedade Padding

A propriedade padding no CSS define o espaçamento interno entre o conteúdo de um elemento e suas bordas, criando um espaço "de dentro" que afasta o conteúdo das bordas do elemento. É uma propriedade abreviada que pode definir os quatro lados do preenchimento: superior, direito, inferior e esquerdo.

Como funciona a propriedade padding

  • Pode ser usada com 1 a 4 valores, aplicados na ordem: top (superior), right (direito), bottom (inferior), left (esquerdo). Exemplo:
    • padding: 20px; - aplica 20px em todos os lados.
    • padding: 10px 15px; - 10px no topo e base, 15px nas laterais.
    • padding: 5px 10px 15px; - 5px no topo, 10px nas laterais, 15px na base.
    • padding: 5px 10px 15px 20px; - valores específicos para cada lado, no sentido horário começando pelo topo.
  • Também é possível definir o padding individualmente com: padding-top, padding-right, padding-bottom, padding-left.
  • Valores negativos não são permitidos para padding.
  • O padding aumenta o tamanho total do elemento, pois é somado à largura e altura definidas, a menos que se use box-sizing: border-box para incluir o padding dentro do tamanho total do elemento.

Criando um botão simples com padding

.botao {
text-decoration: none;
padding:20px;
margin-bottom: 70px;
border-style: double;
border-radius: 50px;
border-color: black;
color: rgb(27, 79, 70);
background-color: rgb(227, 173, 58);
}
.botao:hover {
color:blue;
background-color: violet;
}

Propriedade Width e Height

As propriedades width e height são usadas para definir, respectivamente, a largura e a altura da área de conteúdo de um elemento, principalmente em CSS e programação visual.

Propriedade Width

  • Define a largura da área de conteúdo de um elemento, excluindo margens, bordas e preenchimentos (padding).
  • Pode ser especificada em valores absolutos (pixels, em, rem, etc.), percentuais (em relação ao elemento pai) ou como auto para ajuste automático.
  • As propriedades min-width e max-width podem limitar a largura mínima e máxima, sobrescrevendo o valor de width.
  • Aplica-se a quase todos os elementos, exceto alguns inline e elementos de tabela específicos.

Propriedade Height

  • Define a altura da área de conteúdo de um elemento, também excluindo margens, bordas e preenchimentos.
  • Pode ser especificada em unidades absolutas, percentuais (em relação à altura do elemento pai, se definida) ou como auto.
  • As propriedades min-height e max-height limitam a altura mínima e máxima, sobrepondo o valor de height.
  • Aplica-se a quase todos os elementos, exceto alguns inline e elementos de tabela específicos.

Propriedade Text

A propriedade text em CSS não é uma propriedade única, mas o termo geralmente se refere ao conjunto de propriedades usadas para estilizar textos em páginas web. Essas propriedades controlam a aparência, formatação e disposição do texto.

  • text-aling: Controla como o texto (ou outros conteúdos inline) é alinhado na horizontal dentro do seu contêiner. Pode ser: left(clinha o texto à esquerda), right(clinha o texto à direita), center(centraliza o texto horizontalmente dentro do elemento), justify(Estica as linhas do texto para que as margens esquerda e direita fiquem alinhadas), inherit(O elemento herda o valor de alinhamento do seu elemento pai).
  • direction: Afeta o fluxo do texto e também pode influenciar o layout de elementos, como em contêineres flexíveis, onde a direção do eixo principal pode ser afetada pelo valor de direction. pode ser: ltr ou rtl
  • unicone-bide: Corrige a exibição de textos que misturam idiomas com direções diferentes. Permite sobrescrever o algoritmo padrão do Unicode para manipulação de texto bidirecional, definindo níveis de incorporação e isolamento.
  • color: Define a cor do texto.
  • font-size: Define o tamanho da fonte, podendo usar unidades como px, em, %, pt.
  • text-decoration: Aplica decorações ao texto, como underline (sublinhado), overline (linha acima), line-through (riscado) ou none (sem decoração).
  • text-transform: Altera a capitalização do texto, com valores como uppercase (maiúsculas), lowercase (minúsculas) e capitalize (primeira letra maiúscula).
  • text-indent: Define o espaçamento da primeira linha de uma paragráfo.
  • word-spacing: Ajusta o espaçamento entre palavras.
  • letter-spacing: Ajusta o espaçamento entre letras. Pode ser negativo, fazendo com que as letras se sobrepõem
  • line-height: Ajusta o espaçamento entre linhas. Pode ser negativo, fazendo com que as linhas se sobrepõem

Propriedade Font

Responsável por definir a formatação das fontes utilizadas em uma página web. Ela pode ser usada para configurar várias características do texto, como família, tamanho, estilo e variante da fonte.

  • font-style: Define o estilo da fonte, podendo ser normal (padrão), italic (itálico) ou oblique (obliquo).
  • font-size: Define o tamanho da fonte, aceitando unidades como px, em, %, entre outras.
  • font-weight: Controla o peso da fonte, como normal, negrito, etc.
  • font-family: : Especifica a família da fonte a ser usada. Pode incluir uma lista de fontes em ordem de prioridade, terminando geralmente com uma família genérica como serif, sans-serif ou monospace para garantir uma fonte alternativa caso as anteriores não estejam disponíveis no sistema do usuário.

Propriedade Display

A propriedade display no CSS define como um elemento HTML será renderizado e como ele participará do layout da página. Ela determina se o elemento será tratado como um bloco, inline, flexível, grade, entre outros, e também influencia o layout dos seus elementos filhos.

block:
O elemento é exibido como um bloco, ocupando toda a largura disponível e iniciando em uma nova linha. Exemplo: <div>, <p>. Elementos block iniciam em nova linha e ocupam toda a largura disponível, enquanto inline ficam lado a lado.
inline:
O elemento ocupa apenas o espaço necessário para seu conteúdo, posicionando-se lado a lado com outros elementos inline. Propriedades de largura e altura são ignoradas.
inline-block:
Combina características de block e inline; permite que o elemento fique lado a lado com outros, mas aceita largura e altura definidas.
grid e inline-grid:
Cria um layout baseado em linhas e colunas, possibilitando definir tamanhos e espaçamentos específicos para as células da grade.Criam um sistema de grade para posicionar elementos em linhas e colunas, facilitando layouts complexos e responsivos.
flex e inline-flex:
Ativam o layout flexível, permitindo alinhar e distribuir os elementos filhos em linha ou coluna, com controle avançado de alinhamento e espaçamento.
none:
Remove o elemento da renderização, fazendo com que ele não ocupe espaço algum na página.
contents:
Faz com que o elemento não gere uma caixa própria, mas seus filhos são renderizados como se estivessem diretamente no lugar dele, mantendo estilos como cor e fonte.

Propriedades de largura e altura funcionam em block e inline-block, mas não em inline.

Display flex e grid são amplamente usados para criar layouts modernos e responsivos.

display: none é diferente de visibility: hidden, pois remove o elemento do fluxo da página.

O valor initial redefine o display para o valor padrão do elemento, e inherit faz o elemento herdar o display do seu pai.

Propriedade Position

A propriedade position no CSS define como um elemento será posicionado dentro do documento HTML. Ela determina o método de posicionamento do elemento na página e pode assumir cinco valores principais.

static (padrão):
o elemento é posicionado de acordo com o fluxo normal do documento, sem aceitar as propriedades top, bottom, left e right. É o comportamento padrão de todos os elementos.
relative:
o elemento é posicionado relativamente à sua posição original no fluxo da página. Com esse valor, é possível usar as propriedades top, bottom, left e right para deslocar o elemento em relação à sua posição padrão, sem tirar o elemento do fluxo do documento.
absolute:
o elemento é posicionado em relação ao seu ancestral mais próximo que tenha position diferente de static. Ele é removido do fluxo normal e pode ser posicionado usando top, bottom, left e right em relação a esse ancestral posicionado.
fixed:
o elemento é posicionado em relação à janela do navegador e permanece fixo no lugar mesmo quando a página é rolada. Também aceita top, bottom, left e right para definir sua posição fixa.
sticky:
o elemento alterna entre comportamento relativo e fixo, dependendo da posição da rolagem da página. Ele se comporta como relativo até atingir um ponto definido (exemplo, top: 0), quando então fica fixo naquele ponto.

Além do valor da propriedade position, as propriedades auxiliares top, bottom, left e right são usadas para definir a posição exata do elemento, exceto quando position está em static, que ignora essas propriedades.

Propriedade Overflow

A propriedade overflow do CSS controla como o conteúdo que excede as dimensões de um elemento é exibido, sendo essencial para gerenciar layouts e evitar sobreposições indesejadas.

Valores principais

visible (padrão):
Exibe o conteúdo mesmo que ultrapasse os limites do elemento, podendo sobrepor outros elementos.
hidden:
Corta o conteúdo excedente, ocultando-o sem barras de rolagem,
scroll:
Exibe barras de rolagem (horizontal e vertical) mesmo que o conteúdo não ultrapasse o elemento
auto:
Adiciona barras de rolagem apenas quando necessário, dependendo do navegador

Controle direcional

overflow-x:
Gerencia o excesso de conteúdo horizontalmente (ex: scroll, hidden)
overflow-y:
Controla o excesso verticalmente (ex: auto, hidden)
Sintaxe abreviada:
overflow: [horizontal] [vertical] (ex: overflow: scroll auto para rolagem horizontal fixa e vertical automática)

Propriedade Float

A propriedade float do CSS é usada para posicionar elementos à esquerda ou à direita dentro de um container, permitindo que outros elementos (como texto ou elementos em linha) se organizem ao seu redor. É amplamente utilizada para criar layouts com colunas, imagens flutuantes e interfaces complexas.

Valores principais

left:
Posiciona o elemento à esquerda do container, com conteúdo fluindo à sua direita.
right:
Posiciona o elemento à direita do container, com conteúdo fluindo à sua esquerda.
none (padrão):
Mantém o elemento no fluxo normal, sem flutuação.
inline-start/inline-end (experimentais):
Posicionam o elemento conforme a direção do texto (ex: esquerda para ltr, direita para rtl).

Comportamento e impacto

  • Remoção do fluxo normal: O elemento é retirado do fluxo do documento, podendo sobrepor outros elementos ou colapsar o container pai se não houver altura definida.
  • Alteração do display: Elementos com float diferente de none têm seu display computado como block (ex: inline → block).
  • Compatibilidade com flexbox/grid: float não afeta elementos com display: flex ou display: grid.

Problemas comuns e soluções

  • Colapso do container pai
    • Causa: Elementos flutuantes não contribuem para a altura do container.
    • Solução: Usar overflow: hidden no container ou adicionar um pseudo-elemento com clear: both.
  • Sobreposição de conteúdo
    • Causa: Elementos subsequentes envolvem o flutuante.
    • Solução: Aplicar clear: left, clear: right ou clear: both no elemento que deve ficar abaixo.

Embora ainda seja útil para alinhar imagens e textos, float foi substituído em layouts complexos por Flexbox e CSS Grid, que oferecem maior controle e responsividade.

Propriedade Opacity

A propriedade opacity em CSS especifica o grau de transparência de um elemento, determinando o quanto o conteúdo de fundo atrás desse elemento será visível. Seu valor varia de 0 a 1, onde:

  • 0 significa totalmente transparente (invisível).
  • valores entre 0 e 1 indicam níveis variados de translucidez.
  • 1 representa totalmente opaco (sem transparência).

Essa propriedade afeta o elemento inteiro, incluindo seu conteúdo e elementos filhos, ou seja, tudo dentro do elemento terá a mesma opacidade relativa ao fundo, mesmo que filhos tenham valores diferentes de opacidade. Por isso, se a intenção é deixar apenas o fundo transparente sem afetar o conteúdo, recomenda-se usar a cor de fundo com valor rgba() em vez de opacity.

Especificidade CSS

A especificidade CSS é um algoritmo que determina qual regra de estilo será aplicada a um elemento HTML quando há múltiplas regras conflitantes. Ela funciona como um sistema de "peso" ou "pontuação" que o navegador usa para decidir qual seletor tem prioridade, garantindo que o estilo mais específico seja aplicado.

  1. Seletores de tipo (elemento) e pseudoelementos: menor peso.
  2. Seletores de classe, atributos e pseudoclasses: peso médio.
  3. Seletores de ID: peso alto.
  4. Estilos inline (atributo style): maior peso.

Exemplo prático

  1. Um seletor de elemento como h1 tem especificidade baixa.
  2. Um seletor de classe como .main-heading tem especificidade maior.
  3. Um seletor de ID como #header tem especificidade ainda maior.
  4. Um estilo inline, como style="color: red;", tem a maior especificidade.

Regra Importent

A regra !important em CSS é uma diretiva que força a aplicação de uma determinada declaração de estilo, independentemente da especificidade dos seletores ou da ordem das regras na folha de estilos. Quando uma propriedade CSS é marcada com !important, ela tem prioridade máxima sobre outras declarações conflitantes, exceto se outra declaração também usar !important com um seletor de maior especificidade.

Características da regra !important

  • Prioridade máxima: A declaração com !important sobrescreve todas as outras regras para aquela propriedade, mesmo que tenham seletores com maior especificidade ou estejam declaradas depois na folha de estilos.
  • Uso cauteloso: O uso excessivo de !important pode tornar o CSS difícil de manter, pois cria um código frágil onde outras regras precisam usar !important para sobrescrever, gerando uma "guerra" de prioridades.
  • Combinação com especificidade: Se duas regras conflitantes usam !important, a especificidade dos seletores é usada para decidir qual será aplicada.
  • Não afeta herança: !important atua apenas na regra onde é declarado, não altera o comportamento da herança CSS.

Exemplo prático

p {
color: blue !important;
}
.text {
color: red;
}

Neste caso, o parágrafo terá a cor azul, mesmo que a regra .text apareça depois, porque o !important na regra do p tem prioridade maior.

Boas práticas

  • Evite usar !important a menos que seja realmente necessário, como para corrigir estilos de terceiros ou em situações específicas.
  • Prefira resolver conflitos com a especificidade correta dos seletores e organização do código.
  • Use !important como último recurso para evitar problemas de manutenção e escalabilidade no CSS.

Efeito Gradiente

O efeito gradiente no CSS é uma técnica que cria uma transição suave entre duas ou mais cores em um elemento, podendo ser aplicado em fundos, bordas, textos, entre outros.

Existem dois tipos principais de gradientes no CSS:

  • Gradiente linear: a transição de cores ocorre ao longo de uma linha reta, que pode ter direção e sentido definidos por ângulos ou palavras-chave como to right, to left, to top, to bottom e suas combinações diagonais. Por padrão, o gradiente linear vai de cima para baixo (equivalente a to bottom). É possível usar vários pontos de cor para criar transições mais complexas e também ajustar transparências para suavizar o efeito. Exemplo: background-image: linear-gradient(direção, cor1, cor2, ...);
  • Gradiente radial: a transição de cor ocorre a partir de um ponto central, expandindo-se em círculos ou elipses. Pode-se definir a posição do centro e o formato do gradiente (círculo ou elipse), além das cores e seus pontos de transição. Exemplo: background-image: linear-gradient(to right, green, black);

Para bordas com efeito gradiente, é comum usar a propriedade background-clip combinada com gradientes para criar bordas coloridas e suaves, preservando o arredondamento com border-radius e usando uma borda transparente para separar as áreas.

Além disso, o efeito gradiente pode ser aplicado em textos usando técnicas como background-clip: text com gradientes lineares para criar títulos coloridos e visuais atraentes.

Propriedades Text Shadow e Box Shadow

Text Shadow

A propriedade text-shadow adiciona sombras ao texto, criando efeitos de profundidade e destaque. Ela aceita uma ou mais sombras, que são aplicadas ao texto em camadas, separadas por vírgulas.

text-shadow: deslocamento-x deslocamento-y raio-de-desfoque cor;
  • deslocamento-x: distância horizontal da sombra (valores positivos deslocam para a direita, negativos para a esquerda).
  • deslocamento-y: distância vertical da sombra (valores positivos deslocam para baixo, negativos para cima).
  • raio-de-desfoque (opcional): quanto maior, mais desfocada e suave fica a sombra.
  • cor (opcional, mas recomendado): cor da sombra.

Exemplo

text-shadow: 2px 3px 2px rgba(0,0,0,0.5);

Box Shadow

A propriedade box-shadow adiciona sombras ao redor de elementos, criando a sensação de profundidade e elevação visual. Pode ser aplicada a quase qualquer elemento HTML.

box-shadow: deslocamento-x deslocamento-y raio-de-desfoque raio-de-espalhamento cor;
  • deslocamento-x: distância horizontal da sombra (valores positivos deslocam para a direita, negativos para a esquerda).
  • deslocamento-y: distância vertical da sombra (valores positivos deslocam para baixo, negativos para cima).
  • raio-de-desfoque (opcional): quanto maior, mais desfocada e suave fica a sombra.
  • raio-de-espalhamento (opcional): aumenta ou diminui o tamanho da sombra.
  • cor (opcional, mas recomendado): cor da sombra.
  • inset (opcional): inverte a sombra para dentro do elemento.

Exemplo

box-shadow: 10px 5px 5px black;

Múltiplas sombras: box-shadow: inset 0 0 8px 4px gold, 0 0 8px 1px black;

Propriedade Aplica-se em Função principal Sintaxe básica Permite múltiplas sombras?
text-shadow Texto Sombra nas letras do texto text-shadow: offset-x offset-y blur color; Sim
box-shadow Caixas (elementos) Sombra ao redor do elemento box-shadow: offset-x offset-y blur spread color [inset]; Sim

Se o elemento tiver bordas arredondadas (border-radius), a sombra acompanha o arredondamento.

Exemplos de uso para simular diferentes fontes de luz, sombras internas, elevação ao passar o mouse, e efeitos artísticos são comuns.

Media Queries

Media Queries são uma funcionalidade do CSS que permite aplicar estilos diferentes a uma página web com base em condições específicas do dispositivo ou da tela, como largura, altura, resolução e orientação. Elas são fundamentais para o design responsivo, pois possibilitam que o layout e a aparência do site se adaptem automaticamente a diferentes tamanhos e tipos de dispositivos, sem alterar o conteúdo em si.

Como funcionam as Media Queries

  • Uma media query consiste em um media type (como screen, print ou all) e uma ou mais expressões chamadas media features (como max-width, min-width, orientation), que avaliam características do dispositivo.
  • Se a condição da media query for verdadeira para o dispositivo que está exibindo a página, os estilos CSS dentro dela são aplicados.
  • A sintaxe básica é:
    @media (condição) {
    /* estilos CSS aplicados se a condição for verdadeira */
    }
  • Exemplo para aplicar estilos em telas com largura máxima de 600px:
    @media (max-width: 600px) {
    body {
    background-color: lightblue;
    }
    }

Exemplos de uso prático

  • Ajustar layouts para smartphones, tablets e desktops, modificando a exibição de elementos como barras laterais, menus e colunas.
  • Alterar tamanhos de fonte, cores, margens e outros estilos para melhorar a legibilidade e usabilidade em diferentes dispositivos.
  • Aplicar estilos específicos para orientação retrato ou paisagem, ou para resoluções específicas.

Características comuns usadas em Media Queries

  • width, min-width, max-width
  • height, min-height, max-height
  • orientation (portrait ou landscape)
  • resolution
  • aspect-ratio
  • device-width, device-height

Vantagens das Media Queries

  • Permitem criar layouts flexíveis e adaptativos, melhorando a experiência do usuário.
  • Evitam a necessidade de múltiplas versões do mesmo site para diferentes dispositivos.
  • São suportadas pela maioria dos navegadores modernos desde CSS3.

Flexbox

O Flexbox, ou Flexible Box Layout, é um modelo de layout do CSS3 que facilita a organização e o alinhamento de elementos dentro de um container de forma dinâmica e flexível, adaptando-se ao tamanho disponível e reorganizando os itens conforme necessário.

Conceitos principais do Flexbox:

  • Flex Container: o elemento pai que recebe display: flex ou display: inline-flex, ativando o modo flexbox para seus filhos.
  • Flex Items: os elementos filhos do container flex, que passam a ser dispostos segundo regras flexíveis.
  • Eixos: o layout flex é unidimensional, com um eixo principal (main axis) e um eixo transversal (cross axis). A direção dos itens pode ser controlada pela propriedade flex-direction (por exemplo, row para linha ou column para coluna).
  • Propriedades do container: incluem justify-content (alinhamento no eixo principal), align-items (alinhamento no eixo transversal), flex-wrap (controle de quebra de linha dos itens), entre outras.
  • Propriedades dos itens: como flex-grow (crescimento), flex-shrink (encolhimento), flex-basis (tamanho base), e order (ordem visual dos itens, independente da ordem no HTML)

Benefícios do Flexbox:

  • Layouts responsivos e adaptativos sem a necessidade de usar floats ou posicionamento complexo.
  • Alinhamento fácil e centralização de elementos vertical e horizontalmente.
  • Controle da ordem visual dos elementos sem alterar o HTML.
  • Distribuição dinâmica do espaço disponível entre os itens.

Exemplo básico:

.container {
display: flex;
flex-direction: row; /* padrão: itens em linha */
justify-content: center; /* centraliza itens no eixo principal */
align-items: center; /* centraliza itens no eixo transversal */
}
